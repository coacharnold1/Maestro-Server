Add Admin API v2.0 with Complete Installer

This major update adds a comprehensive system administration interface
and a complete installer for fresh Ubuntu/Debian/Arch installations.

## New Features

### Admin API (Port 5004)
- Real-time system monitoring (CPU, RAM, Disk, Network)
- Library management with NFS/SMB mount support
- Audio device scanning and configuration
- System updates with progress tracking
- System reboot with countdown timer
- MPD library updates with verification

### Complete Installer
- Auto-detects OS (Ubuntu/Debian/Arch)
- Installs MPD + all dependencies
- Configures audiophile-optimized MPD defaults
- Creates systemd services for auto-start
- Sets up sudo permissions
- ~5 minute installation time

### Integration
- Admin API integrates seamlessly with existing Web UI
- Shared configuration storage (~/.config/maestro/)
- Independent services (can run separately)
- Comprehensive documentation
