üìö Common abcde.conf Settings

CDDBMETHOD - Metadata lookup method

cddb: Use CDDB/freedb
musicbrainz: Use MusicBrainz (recommended)

OUTPUTTYPE - Output format

flac: FLAC lossless
mp3: MP3 lossy
ogg: Ogg Vorbis
opus: Opus codec

OUTPUTDIR - Where files are saved

Default: "$HOME/Music"
For MPD: "/var/lib/mpd/music"

OUTPUTFORMAT - File naming pattern

Variables: ${ARTISTFILE}, ${ALBUMFILE}, ${TRACKNUM}, ${TRACKFILE}
Example: '${ARTISTFILE}/${ALBUMFILE}/${TRACKNUM} - ${TRACKFILE}'

ACTIONS - What abcde should do

cddb: Lookup metadata
read: Rip tracks
encode: Encode to format
tag: Write metadata tags
move: Move to output directory
clean: Remove temp files

MAXPROCS - Parallel encoding jobs

Number of CPU cores to use (e.g., 4)

FLACOPTS - FLAC encoding options

-8: Maximum compression (slower)
-5: Balanced (default)
-0: Fastest compression

LAMEOPTS - MP3 encoding options

-b 320: 320kbps constant bitrate
-V 0: Variable bitrate, highest quality
-V 2: Variable bitrate, high quality

EJECTCD - Auto-eject after rip

y: Yes, eject disc when done
n: No, keep disc in drive
