## üí° Tips for Great Screenshots

1. **Theme Selection:**
   - Dark theme: Best for showing UI elements clearly
   - Sunset/Desert: Warm, inviting, great for hero images
   - High Contrast: Excellent for accessibility showcase
   - Terminal: Appeals to tech enthusiasts

2. **Content Selection:**
   - Use recognizable artists (Beatles, Pink Floyd, etc.)
   - Show diverse album art (colors, styles)
   - Include varied metadata (long/short titles)

3. **Composition:**
   - Center important UI elements
   - Leave some breathing room (don't crop too tight)
   - Show enough context to understand the feature

4. **Consistency:**
   - Use same music library across related screenshots
   - Maintain consistent window size
   - Similar zoom levels for comparable features
